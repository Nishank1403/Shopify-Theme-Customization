# Shopify-Theme-Customization

A portfolio-ready Shopify Liquid customization project that showcases a product page template, customer dashboard section, invoice template, and an email template. Built with Liquid, HTML, CSS, and JavaScript to highlight reusable components and responsive UI patterns.

## ✨ Features
- **Product page template** with variant selector, quantity control, badges, and trust messaging.
- **Reusable sections** for product features and customer dashboard panels.
- **Invoice template** that previews cart line items and totals.
- **Order confirmation email template** designed for Shopify notifications.
- **Theme settings** for brand colors and shipping messaging.
- **Responsive layout** with modern UI components.

## 📁 Project Structure
- `layout/theme.liquid` — Main theme layout
- `templates/product.liquid` — Product page template
- `templates/customers/account.liquid` — Customer dashboard page
- `templates/page.invoice.liquid` — Invoice page template
- `sections/*` — Reusable Liquid sections
- `snippets/*` — Shared UI snippets
- `assets/styles.css` — Base styles
- `assets/theme.js` — UI interactions
- `email-templates/order-confirmation.liquid` — Email example
- `docs/index.html` — GitHub Pages preview

## 🚀 How to Use
1. Upload the repository as a Shopify theme or add files to an existing theme.
2. Add the **Product Features** section to your product page and configure the blocks.
3. Create a page in Shopify and assign the **invoice** template to preview invoices.
4. Copy the email template into Shopify Notifications to customize order confirmations.
